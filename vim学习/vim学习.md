# Vim教程打开

命令为`vimtutor`。



# Vim使用

```shell
h # 左移动
j # 向下移动一行
k # 向上移动一行
l # 向右移动

:q! # 放弃所有改动，退出

x # Normal模式下，删除挂表所在位置的字符
i # 进入insert模式，在光标前一个位置开始插入
a # 进入insert模式，在光标后一个位置开始插入
A # 进入insert模式，在行尾开始插入
:w # 保存
:wq # 保存并退出
<ESC> # 回到normal模式，或者撤销一个不想输入或者部分完整的命令

dw # 从光标处删除至一个单词的末尾
d$ # 从光标处删除至行末尾

改变文本的命令通常由一个操作符和一个动作构成，比如删除操作符d的删除命令格式
d motion
d是删除操作符，motion是操作符的操作对象，例如
w # 从光标当前位置直到下一个单词起始处，不包括下一个单词的第一个字符
e # 从光标当前位置直到当前单词末尾，包括当前单词的最后一个字符
$ # 从光标当前位置到当前行末
可以直接用上面的motion命令来移动光标

在动作前输入数字，可以使该动作重复多次
2w # 使光标向前移动两个单词
3e # 使光标向前移动到第三个单词的末尾
0 # 光标回到行首

数字也可以与操作符配合
d2w # 删除两个单词

dd # 删除整行
2dd # 删除两行

u # 撤销最后执行的一次命令
U # 恢复光标所在行到其原始状态
CTRL-R # 撤销 撤销命令

p # 将复制、剪切的内容复制到光标所在行的下一行

r # 当需要替换一个字符，将光标移动到要替换的位置

c # change的缩写，搭配motion完成替换操作，例如
cw # 删除从当前光标所在位置到下一个单词首字母，并进入insert模式
ce # 删除从当前光标所在位置到单词末尾，并进入insert模式
c [number] motion
c$ #删除从当前光标所在位置到行末尾，并进入insert模式

CTRL-G(g可以大写也可以小写) # 显示当前编辑文件中当前光标所在位置以及文件状态信息
gg # 光标移动到文件第一行
G # 光标移动到文件最后一行
[number] G # 光标移动到第number行

/ 字符串 # 向后查找字符串
? 字符串 # 向前查找字符串
n # 查找同上一次的字符串
N # 按相反方向查找同上一次的字符串
CTRL-O # 回到跳转之前的位置（跳转到跳转列表中上一个位置（即更早的记录）。）
CTRL-I # 跳转到跳转列表中下一个位置（即更近的记录）。
# 示例
# 从行1跳转到行100（例如输入:100）。
# 再跳转到行200（输入:200）。
# 按Ctrl+O：返回行100。
# 再按Ctrl+O：返回行1。
# 按Ctrl+I（或Tab）：跳回行100。
# 再按Ctrl+I：跳回行200。

: [number] # 跳转到number行

% # 用于括号匹配。当光标位于(、[、{处时，nromal模式下，输入%，则光标跳转到与之匹配的)、]、}处。再输入一次%则跳转回去

:s/old/new # 将光标所在行中第一个old字符串替换为new字符串
:s/old/new/g # 将光标所在行所有old字符串替换为new字符串
:%s/old/new/g # 将文件中所有old字符串替换为new字符串
:%s/old/new/gc # 找到文件中每个匹配字符串，并对每个匹配字符串提示是否需要进行替换
:[number1], [number2]s/old/new/g # 将文件中number1行与number2行之间进行替换操作

:![shell 命令] # 在VIM内执行外部命令，例如:!ls
:w FILENAME # 将当前文档另存为FILENAME

v # normal模式下输入v则进入选择模式（可视模式）（visual），通过移动光标来选择选中范围，选中之后，输入:，则屏幕底部出现:'<,'>，这时输入w TEST，则将选中的内容领存在名为TEST的文件中
# 也可以选中内容，然后按d，删除选中的内容
v [motion] # 结合motion来移动选中内容
:r FILENAME # 提取FILENAME文件中的内容，将其插入到光标位置。也可以:r !ls，则将当前路径下的文件吗插入当前文件

o # 在光标下方开启新的一行，并进入insert模式
O # 在光标上方开启新的一行，并进入insert模式

R # 进入替换模式，此时输入的内容将替换光标所在位置的内容，按ESC退出到normal模式
y # 复制操作符，和删除操作符d、change操作符c是一个性质。可以现在visual模式选中需要复制的内容，让后将光标移动到需要插入的位置，按p粘贴。y可以结合motion使用，比如yw复制一个单词

# 设置类命令
:set ic # 即设置Ignore Case，忽略大小写，这样使用/或者?搜索的时候，将忽略大小写进行字符串匹配查找
:set hlsearch # 设置高亮，查找到的内容将被高亮，可以简写为:set hls
:set is # 设置incsearch，即增量搜索，这样在搜索的时候，每输入一个字符都会进行一次搜索，而不是是等回车才进行搜索
:nohlsearch # 取消高亮（因为搜索后，匹配的字符串会一直高亮，可以用这个命令取消高亮）

按F1键或者输入:help # 打开VIM的在线帮助系统
:q # 退出帮助系统（其实就是关闭这个文件）
CTRL-W # 窗口之间切换
:help w # 给出与w这个motion相关的帮助信息

# 如果需要开启VIM的功能特性，可以在vimrc这个配置文件设置
# $VIMRUNTIME/vimrc_example.vim这个文件有vimrc的示例配置

CTRL-D # 在:部分命令 时，按CTRL-D可以给出可能匹配的命令列表
按TAB键 # 自动补全（如果只有唯一匹配）
```
